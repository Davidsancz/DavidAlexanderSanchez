GitHub es una plataforma de alojamiento de código fuente y control de versiones basada en Git, que permite a los desarrolladores gestionar proyectos, colaborar en el desarrollo de software, y compartir sus repositorios de código con otros. GitHub ofrece una interfaz fácil de usar para gestionar repositorios de Git y proporciona herramientas adicionales para la colaboración, como el control de versiones, seguimiento de problemas (issues), integración continua (CI/CD), y más.
Fundadores de GitHub
